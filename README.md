# lightclock
高速な針の計算
# 🚀 超高速時計シミュレーション (Pygame)

このプロジェクトは、Python の `pygame` ライブラリを使用した、時計の針の先端速度をシミュレーションするアプリケーションです。

回転周期を繰り返し分割することで、針の先端速度が**光速（c）を超える**仮想的な状況を扱います。

---

## 💡 シミュレーションの主な機能

* **古典力学モデル:** 速度 $v = r\omega$（速度＝半径 × 角速度）に基づき、速度の増大をシミュレートします。特殊相対性理論は考慮していません。
* **光速の強調:** 針の先端速度が $299,792,458 \text{ m/s}$（光速）を超えると、針の色が**マゼンタ色**に変化します。
* **データ表示:** 画面右側のパネルに、各針の周期、時速、光速に対する比率が詳細に表示されます。

---

## 💻 使い方と操作方法

### 1. 動作環境の準備

このプログラムを実行するには、**Python 3.x** と **Pygame** ライブラリが必要です。

```bash
# Pygameのインストール
pip install pygame
```
## 2. 操作方法

| 操作対象 | 機能 | 詳細 |
| :--- | :--- | :--- |
| **ボタン A (追加)** | 新しい針の生成 | クリックするたびに、現在の最も速い針の周期を $1/60$ に分割した、**さらに細かい（速い）針**を追加します。 |
| **ボタン B (削除)** | 針の削除 | 最も新しく追加された（最も速い）針を削除します。**秒針（基本）は削除できません。** |
| **マウスホイール**| 情報パネルのスクロール | マウスカーソルを右側の情報パネルに合わせた状態でホイールを回すと、**超高速になった針のデータ**を上下に確認できます。 |
