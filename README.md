# 超高速時計シミュレーション (Pygame)

Pythonを使用した、時計の針の先端速度をシミュレーションするアプリケーションです。

回転周期を繰り返し分割することで、針の先端速度が**光速（c）を超える**仮想的な状況を扱います。

---

## シミュレーションの主な機能
* **光速の強調:** 針の先端速度が $299,792,458 \text{ m/s}$（光速）を超えると、針の色が**マゼンタ色**に変化します。
* **データ表示:** 画面右側のパネルに、各針の周期、時速、光速に対する比率が詳細に表示されます。

---
